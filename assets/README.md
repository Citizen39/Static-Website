# assets

Place `hero-code.png` here.

It's the faded code-on-screen photo used as the hero background texture
(rendered at `opacity: 0.14` with `mix-blend-mode: screen` over a radial cyan
glow). The hero works without it — you just lose the texture layer — but for
full fidelity, save the image from the handoff package as:

```
assets/hero-code.png
```
